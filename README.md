# 🔢 Python Prime Checker

A beginner-friendly Python project that demonstrates how to work with prime numbers.

## 📂 Programs Included

### 1. Prime Number Checker
Checks whether a given number is prime or not.

### 2. First N Prime Numbers
Generates the first N prime numbers.

## 🧠 Concepts Used

- Variables
- User Input
- Boolean Variables
- if-else Statements
- for Loop
- while Loop
- break Statement
- Modulus Operator (%)

## ▶️ How to Run

### Prime Number Checker

```bash
python "prime checker.py"
```

### First N Prime Numbers

```bash
python Nprimenumbers.py
```
